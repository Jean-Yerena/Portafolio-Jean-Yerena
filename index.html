Acerca de mí

Hola, ¡bienvenido a mi página personal! Mi nombre es Jean Yerena, y soy un desarrollador frontend junior apasionado por crear experiencias digitales atractivas y accesibles.

Siempre he tenido una gran curiosidad por la tecnología y el diseño. Me encanta trabajar con HTML, CSS y JavaScript para crear páginas web y aplicaciones que no solo sean funcionales, sino también hermosas y fáciles de usar.

he estado trabajando duro para mejorar mis habilidades y adquirir nuevas herramientas y conocimientos, estoy emocionado por la oportunidad de aprender y crecer en esta industria en constante evolución.

Proyectos

Los siguientes proyectos exhiben mis habilidades y experiencia a través de ejemplos reales, muestran mi capacidad para resolver problemas, trabajar con diferentes tecnologías y administrar proyectos de manera efectiva. Cada proyecto se describe brevemente, cuentan con un enlace que los lleva al proyecto.

Cotizador de criptomonedas

Un sitio web convertidor de monedas a criptomonedas, hecho con JavaScript y una API, permite a los usuarios convertir valores entre diferentes monedas y criptomonedas en tiempo real. La página web tiene una interfaz simple, donde los usuarios pueden seleccionar la moneda o criptomoneda que desean convertir.

Buscador de recetas

Esta página web para buscar recetas, hecha con JavaScript y una API, es una herramienta útil para las personas que buscan recetas de cocina de manera rápida y eficiente, y para aquellos que buscan nuevas ideas e inspiración para cocinar.

Portafolio web

Sitio web para un ingeniero mecánico, está diseñado para ser fácil de navegar y presentar la información de una manera clara y concisa. La combinación de imágenes de calidad y diseño sencillo permite que el ingeniero muestre su experiencia y habilidades de una manera visualmente atractiva.

Administrador de clientes

La página web está diseñada para ser fácil de usar y eficiente en la administración de la información de los clientes. La combinación de JavaScript y IndexedDB permite que la página se cargue rápidamente y funcione sin problemas, lo que la convierte en una herramienta práctica para cualquier negocio que necesite administrar información de clientes.

App para saber el clima

La página web proporciona información detallada y actualizada sobre el clima de una ubicación específica. La combinación de JavaScript y una API de pronóstico del clima permite que la página se cargue rápidamente y brinde información precisa y confiable para los usuarios.
